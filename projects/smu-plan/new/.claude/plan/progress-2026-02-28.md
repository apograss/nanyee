## 技术栈备忘
- Next.js 15 App Router + Prisma + SQLite
- JWT auth (jose): access_token 15min + refresh_token 30d
- CSS Modules + Neo-Brutalism 设计 (3px borders, brand #E8652B, paper bg #FDF0D5)
- 原子组件: NeoButton(variant/size/isLoading), NeoInput(label/error)
- Admin守卫: requireAdmin(req), API格式: { ok, data } / { ok: false, error: { code, message } }
- 所有admin操作写AuditLog，mutation用$transaction包裹
